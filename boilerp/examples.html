Modern Animated Glowing Border - Code Examples

1. Basic Box

Basic Box
CSS Code 
#basic-box {
    background: #142d4a;
    border-radius: 10px;
    width: 150px;
    height: 150px;
}
This is our starting point - a simple box with a dark background and rounded corners.
Works in all browsers

2. Static Gradient Border

Static Gradient
CSS Code 
#static-gradient {
    background: #142d4a;
    border-radius: 10px;
    width: 150px;
    height: 150px;
    position: relative;
}

#static-gradient::before {
    content: "";
    position: absolute;
    height: 100%;
    width: 100%;
    background: conic-gradient(
        from 0deg at center,
        #00c3ff,
        #4d0199,
        #6300c6,
        #009dcd
    );
    border-radius: inherit;
    z-index: -1;
    padding: 2px;
    top: -2px;
    left: -2px;
}
Adding a static conic gradient using the ::before pseudo-element to create a colorful border.
Works in modern browsers

4. Rotating (Transform)

Rotating
CSS Code 
#rotate-transform {
    background: #142d4a;
    border-radius: 10px;
    width: 150px;
    height: 150px;
    position: relative;
}

#rotate-transform::before,
#rotate-transform::after {
    content: "";
    position: absolute;
    height: 100%;
    width: 100%;
    background: conic-gradient(
        from 0deg at center,
        #00c3ff,
        #4d0199,
        #6300c6,
        #00c3ff
    );
    border-radius: inherit;
    z-index: -1;
    padding: 2px;
    top: -2px;
    left: -2px;
    animation: rotate 3s linear infinite;
}

#rotate-transform::after {
    filter: blur(10px);
}

@keyframes rotate {
    from {
        transform: rotate(0deg);
    }
    to {
        transform: rotate(360deg);
    }
}
Animating the gradient by rotating the entire pseudo-element using CSS transforms.
Works in all modern browsers

5. Rotating (CSS Variables)

Rotating
CSS Code 
@property --angle {
    syntax: "<angle>";
    initial-value: 0deg;
    inherits: false;
}

#rotate-property {
    background: #142d4a;
    border-radius: 10px;
    width: 150px;
    height: 150px;
    position: relative;
}

#rotate-property::before,
#rotate-property::after {
    content: "";
    position: absolute;
    height: 100%;
    width: 100%;
    background: conic-gradient(
        from var(--angle) at center,
        #00c3ff,
        #4d0199,
        #6300c6,
        #00c3ff
    );
    border-radius: inherit;
    z-index: -1;
    padding: 2px;
    top: -2px;
    left: -2px;
    animation: rotate-angle 3s linear infinite;
}

#rotate-property::after {
    filter: blur(10px);
}

@keyframes rotate-angle {
    to {
        --angle: 360deg;
    }
}
Using the @property rule to animate CSS custom properties. This is a newer approach that might not be supported in all browsers.

Browser Support Information

If the rotating effects (Techniques 5 and 6) aren't working, it's likely due to limited browser support for the @property rule.

Browser @property Support Conic Gradient Support
Chrome 85+ 69+
Firefox Not Supported 83+
Safari Not Supported 12.1+
Edge 85+ 79+

Solution: If @property isn't supported, use the transform method (Technique 4) which has broader browser support.
