Glowing Border Techniques - Step by Step

1. Basic Box

Basic Box
.basic-box {
  background: #142d4a;
  border-radius: 10px;
  height: 150px;
  width: 150px;
}
This is our starting point - a simple box with a dark background and rounded corners.
Should work in all browsers

2. Static Gradient Border

Static Gradient
.static-gradient::before {
  content: "";
  position: absolute;
  background: conic-gradient(...);
  border-radius: inherit;
  z-index: -1;
  padding: 2px;
}
Adding a static conic gradient using the ::before pseudo-element. This creates a colorful border around the box.
Should work in modern browsers

3. Gradient with Blur

Static + Blur
.static-blur::before,
.static-blur::after {
  /* Same gradient as before */
}
.static-blur::after {
  filter: blur(10px);
}
Adding a blurred duplicate using the ::after pseudo-element to create a glowing effect.
Should work in modern browsers

4. Rotating (Transform)

Rotating
.rotate-transform::before,
.rotate-transform::after {
  /* Same gradient as before */
  animation: rotate 3s linear infinite;
}
@keyframes rotate {
  to { transform: rotate(360deg); }
}
Animating the gradient by rotating the entire pseudo-element using CSS transforms.
Should work in all modern browsers

5. Rotating (CSS Variables)

Rotating
@property --angle {
  syntax: "<angle>";
  initial-value: 0deg;
  inherits: false;
}

.rotate-property::before {
  background: conic-gradient(from var(--angle) ...);
  animation: rotate-angle 3s linear infinite;
}
@keyframes rotate-angle {
  to { --angle: 360deg; }
}
Using the @property rule to animate CSS custom properties. This is a newer approach that might not be supported in all browsers.
Might not work in some browsers

6. Final Effect

Final Effect
.final-effect::before,
.final-effect::after {
  background: conic-gradient(from var(--angle) ...);
  animation: final-animation 2s linear infinite;
}
.final-effect::after {
  filter: blur(10px);
}
@keyframes final-animation {
  to { --angle: 360deg; }
}
The complete effect combining the rotating gradient with the blur effect.
Might not work in some browsers

Browser Support Information

If the rotating effects (Techniques 5 and 6) aren't working, it's likely due to limited browser support for the @property rule.

Browser @property Support Conic Gradient Support
Chrome 85+ 69+
Firefox Not Supported 83+
Safari Not Supported 12.1+
Edge 85+ 79+

Solution: If @property isn't supported, use the transform method (Technique 4) which has broader browser support.
