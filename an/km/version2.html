DeepSite Effect

Large liquid blobs morphing behind dark translucent layers

Layer 1: Blobs

Large gradient divs with heavy blur and morphing animations

Layer 2: Overlay

Dark translucent mask (85% opacity) mutes the colors

Layer 3: Noise

Grain texture overlay for tactile depth

/* The trick is the layering order:
   1. Blobs (z-index: 0, fixed, heavy blur)
   2. Dark overlay (z-index: 0, rgba(10,10,10,0.85))
   3. Noise (z-index: 0, mix-blend-mode: overlay)
   4. Content (z-index: 10) */
