How this carousel works

  • Uses CSS-only scroll buttons (::scroll-button) for navigation
  • Implements anchor positioning (anchor-name, position-anchor) for button placement
  • Features scroll markers (::scroll-marker) for position indicators
  • Utilizes scroll snapping (scroll-snap-type) for precise card alignment
  • Includes smooth scrolling (scroll-behavior: smooth) for better UX
  • Toggle between dark and light mode using the switch
Note: This demo uses cutting-edge CSS features that may not be supported in all browsers yet. For best results, try in the latest version of Chrome or Safari.
