Click on any button below to navigate to the index.html file in the corresponding subdirectory.

How to Add New Directories

  1. Create a new subdirectory in your file system (e.g., new-folder)
  2. Add an index.html file inside that subdirectory
  3. Open this file in your editor and find the directories array in the JavaScript section
  4. Add your new directory name to the array, following the same format
  5. Save the file and refresh this page - a new button will appear automatically!

Example: To add a directory named "analytics", add "analytics" to the array.
