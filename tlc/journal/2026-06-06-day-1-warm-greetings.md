# Day 1 — Warm greetings

## Learning objective

Start a friendly Tagalog chat without sounding formal or stiff. Today is about a few safe, common greeting chunks you can actually use.

## Phrase set

| Tagalog / Filipino | Meaning | Tone |
|---|---|---|
| Kumusta? | How are you? / What’s up? | Safe, common |
| Musta? | How are you? / What’s up? | More casual chat spelling |
| Uy! | Hey! | Friendly attention-getter |
| Anong balita? | What’s new? | Casual check-in |
| Okay ka lang? | Are you okay? | Caring check-in |
| Ayos lang. | I’m okay / all good. | Casual reply |
| Salamat! | Thanks! | Always useful |

## Tone notes

- **Kumusta?** is the clear beginner-safe version.
- **Musta?** is shorter and common in casual messages.
- **Uy!** is like “hey!” Use it with people you are already friendly with.
- **Anong balita?** literally asks “what news?” but works like “what’s up / what’s new?”
- Avoid trying to sound too slangy on day one. Friendly and simple beats forced slang.

## Mini chat examples

```text
You: Uy, musta?
Friend: Okay lang. Ikaw?
You: Ayos lang, salamat!
```

```text
You: Kumusta? Anong balita?
Friend: Wala naman haha.
You: Ayos lang 😄
```

## 2-minute practice

1. Say these aloud three times: `Kumusta?`, `Musta?`, `Anong balita?`, `Ayos lang.`
2. Write one message you would actually send to a Filipino friend using **Uy** or **Musta**.
3. Keep it short. Example: `Uy, musta? Anong balita?`

## Reflection question

Which greeting feels most natural for you to send today: **Kumusta?**, **Musta?**, or **Uy, musta?** Why?
